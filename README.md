# Life Management App (LMA)

## 📋 Life Management App Plan

### Core Features
1. **Task Management (Planner Node)**
   - Kanban-style task board
   - CRUD operations
   - Status tracking (ToDo, InProgress, Completed)
   - Priority and labels
   - Deadline management

2. **Notes System** ✓ (Already implemented)
   - Markdown editor
   - SQLite storage

3. **Weather Integration** ✓ (Already implemented)
   - Location-based forecasts
   - 7-day forecast

4. **AI Assistant (AI Node)**
   - Voice/text commands
   - Task management via natural language
   - Joke of the day integration

5. **Pomodoro Timer**
   - Work/rest intervals
   - Session tracking
   - Task-linked timers

### Architecture
- **Framework:** .NET MAUI 9.0
- **Database:** SQLite (Entity Framework Core)
- **Pattern:** MVVM (Model-View-ViewModel)
- **Services:** Dependency Injection
